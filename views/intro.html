展示

iconfont 平台会分类展示业务中使用的 icon 图标，包括 pc 平台侧 和 手机移动侧。

下载

新业务中需要使用 iconfont 平台上的某个或者某些图标时，可以选择需要的图标，生成字体文件，避免加载整个字体文件（文件体积大）。具体操作步骤是：

  • 1. 选择需要使用的 icon 图标，选中的 icon 图标右上角有打勾
  • 2. 点击下载，会将选中的 icon 图标生成字体文件下载到本地，同时生成 demo

上传

新的 icon 图标从设计师那里过来时，可以点击导航上的上传，将 svg 文件拖拽到框内（svg制作规范），完成 svg 的上传，iconfont 平台接受 svg 后，将 svg 导入到数据库中，并显示在平台上。
